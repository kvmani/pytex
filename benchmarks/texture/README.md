# Texture Benchmarks

This directory stores benchmark manifests for texture-domain validation programs that do not fit cleanly under EBSD-only or diffraction-only headings.

Current focus:

- XRDML pole-figure import and ODF reconstruction from vendor-style texture measurements
- explicit import normalization policy and inversion-diagnostic coverage for those measurements
- open LaboTex `PPF` and `EPF` pole-figure import plus external multi-material validation runs for Cu, Fe, and Al
