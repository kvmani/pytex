# EBSD Benchmarks

This directory holds benchmark manifests for implemented EBSD workflows.

Current scope:

- regular-grid and graph-backed neighbor generation
- multiphase phase selection and phase-aware texture extraction
- KAM on regular grids and irregular coordinate graphs
- grain segmentation, GROD, boundary extraction, and cleanup with cross-phase guards

These artifacts are still lightweight, but the benchmark identity and schema surface are now stable.
